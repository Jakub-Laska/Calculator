Calculator app by JL

    Calculator Info

    Hotkeys:

    • I: Toggle info screen
    • L: Toggle light/dark mode
    • M: Mute/unmute sounds
    • ESCAPE: Clear all
    • C: Clear current number
    • BACKSPACE: Remove last digit
    • 0-9: Enter numbers
    • +, -, *, /: Operators
    • .: Add/remove decimal
    • Shift + -: Toggle negation (+/-)
    • = or ENTER: Calculate result

    Functionality:

    • Supports two-number operations with live result display
    • Sounds for buttons, operators, and actions
    • Responsive font resizing based on screen width and input length
    • 16-digit input limit per number
    • Light/dark mode with button or hotkey
    • Period button toggles decimal on/off
    • Negation key toggles positive/negative numbers
